Created this project to learn how to build a full stack web application with NodeJS and Express

Functionality

  • sign up
  • login
  • view messages
  • become member to see usernames on messages
  • delete messages when user is admin

Technologies used

  • NodeJS
  • Express
  • Mongodb

Showcase of project

Authentication

I used passportjs username & password for authentication. This was my first full stack project that has authentication, so I did not really know where to start. I got passportjs recommended so I tried implementing it. I struggled hard for 2 weeks because the documentation of passportjs is very bad. Some crucial steps that I got help with weren't even on the documentation. But after 2 weeks I got it working.

Authorization

For authorization I used the serialize and deserialize functions from passportjs. This uses sessions to keep the user logged in.

Views

For frontend I used EJS because it is very similar to html. I loaded the data from the server in the EJS views.
