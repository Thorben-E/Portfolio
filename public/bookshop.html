Showcase of project

Design choices

First I started building the navigation with react router. I created my routes and added react components as elements. I also prop drilled the cart data and cart functionality through these routes.

To make the cart work I used useState to create an array where I can add books. Next I create a function to add items to the cart and a function to remove items from the cart. Then I prop drilled these functions and the cart array to the components that needed it. (you can see this in the picture above)
